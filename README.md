##Agriculture Data Analysis using SQL (ICRISAT Dataset)##

## Project Overview
This project focuses on analyzing district-level agricultural data from the ICRISAT dataset using SQL. The goal is to understand crop production patterns, yield efficiency, and regional performance across different states and years in India.

---

## Objectives
- Identify states with high cultivation area but low yield (inefficient regions)
- Analyze year-wise trends for major crops like Rice, Wheat, and Cotton
- Compare crop productivity across states and districts
- Evaluate agricultural efficiency using yield (production per area)
- Highlight regions requiring policy intervention
- 
## Tools & Skills Used
- SQL (Joins, Aggregations, GROUP BY, CASE, CTEs)
- Data Cleaning & Transformation
- Analytical Thinking & Problem Solving

---

## Dataset Description
- Source: ICRISAT
- Level: District-Year
- Key Metrics:
  - Area (in 1000 hectares)
  - Production (in 1000 tons)
  - Yield (Kg per hectare)
- Covers multiple crops including Rice, Wheat, Maize, Cotton, Oilseeds, etc.

## Key Analysis Performed

1. Identified states with high rice cultivation area but low yield
2. Analyzed year-wise production trends of major crops
3. Compared state-wise and district-wise crop performance
4. Evaluated cotton production trends in Telangana
5. Identified top-performing states and districts for crop production
6. Calculated yield per hectare and per acre for better efficiency comparison


## Key Insights
- High production does not always indicate efficiency; yield provides better insights
- Some states have large cultivation areas but relatively low productivity
- Crop performance varies significantly across regions and years
- Identifying low-yield regions can help in targeted agricultural improvements
